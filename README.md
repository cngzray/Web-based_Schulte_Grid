舒尔特方格为经典专注力训练与检测工具，方格内随机无序排布数字 1—25。测试需按从小到大顺序，依次快速找出全部数字，手指点指或目光定位均可，全程保持专注、不跳数、不回看。以完整读完 1 至 25 的用时作为评判标准，用时越短，代表视觉专注力、注意力集中度、视觉反应速度与抗干扰能力越强；耗时过长、频繁卡顿、找数混乱，则说明注意力容易分散，视觉追踪与专注稳定性较弱。
